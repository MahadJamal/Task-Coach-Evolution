isselected(self, item)

source code 

When this viewer is in aggregation mode, curselection returns the actual underlying Effort objects instead of aggregates. This is a problem e.g. when exporting only a selection, since items we're iterating over (aggregates) are never in curselection(). This method is used instead. It just ignores the overridden version of curselection.
