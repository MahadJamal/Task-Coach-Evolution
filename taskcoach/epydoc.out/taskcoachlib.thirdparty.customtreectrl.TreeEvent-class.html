Class TreeEvent

TreeEvent is a special class for all events associated with tree controls.

:note: Not all accessors make sense for all events, see the event descriptions below.

Veto(self)

source code  

Prevents the change announced by this event from happening.

:note: It is in general a good idea to notify the user about the reasons
 for vetoing the change because otherwise the applications behaviour (which
 just refuses to do what the user wants) might be quite surprising.

Allow(self)

source code 

This is the opposite of Veto: it explicitly allows the event to be processed. For most events it is not necessary to call this method as the events are allowed anyhow but some are forbidden by default (this will be mentioned in the corresponding event description).
