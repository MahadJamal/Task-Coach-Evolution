Method Details [hide private]

domainObjectsToView(self)

source code 

Return the domain objects that this viewer should display. For global viewers this will be part of the task file, e.g. self.taskFile.tasks(), for local viewers this will be a list of objects passed to the viewer constructor.

onSelect(self, event=None)

source code 

The selection of items in the widget has been changed. Notify our observers.

curselectionIsInstanceOf(self, class_)

source code 

Return whether all items in the current selection are instances of class_. Can be overridden in subclasses that show only one type of items to simply check the class.

isselected(self, item)

source code 

Returns True if the given item is selected. See EffortViewer for an explanation of why this may be different than 'if item in viewer.curselection()'.

select_all(self)

source code 

Select all items in the presentation. Since some of the widgets we use may send events for each individual item (!) we stop processing selection events while we select all items.

bitmap(self)

source code
