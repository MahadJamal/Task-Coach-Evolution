Class BasicFileKeyring

source code

    object --+    
             |    
KeyringBackend --+
                 |
                BasicFileKeyring

BasicFileKeyring is a file-based implementation of keyring.

This keyring stores the password directly in the file and provides methods which may be overridden by subclasses to support encryption and decryption. The encrypted payload is stored in base64 format.
