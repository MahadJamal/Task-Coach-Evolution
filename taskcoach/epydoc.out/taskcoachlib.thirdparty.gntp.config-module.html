Module config

source code

The gntp.config module is provided as an extended GrowlNotifier object that takes advantage of the ConfigParser module to allow us to setup some default values (such as hostname, password, and port) in a more global way to be shared among programs using gntp

Code duplication is bad, but for right now I have copied the mini() function from the gntp.notifier class since I do not know of an easy way to reuse the code yet fire using the copy of GrowlNotifier in this module
