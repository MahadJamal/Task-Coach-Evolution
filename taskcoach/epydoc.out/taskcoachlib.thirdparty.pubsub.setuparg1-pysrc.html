 1  ''' 
 2  Setup pubsub for use of *arg1* messaging protocol. Without this  
 3  module, the default is the more powerful *kwargs* protocol. The  
 4  *arg1* protocol is identical to the legacy messaging protocol from  
 5  first version of pubsub (when it was still part of wxPython) and  
 6  is *deprecated*. This module is therefore *deprecated*.  
 7   
 8  Usage: Import this file before the first 'from pubsub import pub'  
 9  statement to make pubsub use the *arg1* messaging protocol:: 
10   
11      from pubsub import setuparg1 
12      from pubsub import pub 
13   
14  Note that once :mod:pub has been imported, the messaging protocol 
15  must NOT be changed.  
16   
17  If you want to migrate an application from 'arg1' to 'kwargs' 
18  style messaging, this module provides a function,  
19  :func:enforceArgName(), which can help you in this endeavour.
