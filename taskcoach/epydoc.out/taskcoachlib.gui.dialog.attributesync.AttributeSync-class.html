Class AttributeSync

source code

object --+
         |
        AttributeSync

Class used for keeping an attribute of a domain object synchronized with a control in a dialog. If the user edits the value using the control, the domain object is changed, using the appropriate command. If the attribute of the domain object is changed (e.g. in another dialog) the value of the control is updated.
