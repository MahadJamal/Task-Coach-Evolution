Module aui_utilities

source code

This module contains some common functions used by :mod:`lib.agw.aui` to manipulate colours, bitmaps, text, gradient shadings and custom dragging images for :class:`~lib.agw.aui.auibook.AuiNotebook` tabs.

Date: 31 March 2009

Author: Andrea Gavana <andrea.gavana@gmail.com>
