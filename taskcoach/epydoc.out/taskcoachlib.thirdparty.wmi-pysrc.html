   1  """
 
   2  Windows Management Instrumentation (WMI) is Microsoft's answer to
 
   3  the DMTF's Common Information Model. It allows you to query just
 
   4  about any conceivable piece of information from any computer which
 
   5  is running the necessary agent and over which have you the
 
   6  necessary authority.
 
   7  
 
   8  Since the COM implementation doesn't give much away to Python
 
   9  programmers, I've wrapped it in some lightweight classes with
 
  10  some getattr / setattr magic to ease the way. In particular:
 
  11  
 
  12  * The :class:`_wmi_namespace` object itself will determine its classes
 
  13    and allow you to return all instances of any of them by
 
  14    using its name as an attribute::
 
  15  
 
  16      disks = wmi.WMI ().Win32_LogicalDisk ()
 
  17  
 
  18  * In addition, you can specify what would become the WHERE clause
 
  19    as keyword parameters::
 
  20  
 
  21      fixed_disks = wmi.WMI ().Win32_LogicalDisk (DriveType=3)
 
  22  
 
  23  * The objects returned by a WMI lookup are wrapped in a Python
 
  24    class which determines their methods and classes and allows
 
  25    you to access them as though they were Python classes. The
 
  26    methods only allow named parameters::
 
  27  
 
  28      for p in wmi.WMI ().Win32_Process (Name="notepad.exe"):
 
  29        p.Terminate (Result=1)
 
  30  
 
  31  * Doing a print on one of the WMI objects will result in its
 
  32    `GetObjectText\_` method being called, which usually produces
 
  33    a meaningful printout of current values.
 
  34    The repr of the object will include its full WMI path,
 
  35    which lets you get directly to it if you need to.
 
  36  
 
  37  * You can get the associators and references of an object as
 
  38    a list of python objects by calling the associators () and
 
  39    references () methods on a WMI Python object::
 
  40  
 
  41      for p in wmi.WMI ().Win32_Process (Name="notepad.exe"):
 
  42        for r in p.references ():
 
  43          print r
 
  44  
 
  45    ..  note::
 
  46        Don't do this on a Win32_ComputerSystem object; it will
 
  47        take all day and kill your machine!
 
  48  
 
  49  
 
  50  * WMI classes (as opposed to instances) are first-class
 
  51    objects, so you can get hold of a class, and call
 
  52    its methods or set up a watch against it::
 
  53  
 
  54      process = wmi.WMI ().Win32_Process
 
  55      process.Create (CommandLine="notepad.exe")
 
  56  
 
  57  * To make it easier to use in embedded systems and py2exe-style
 
  58    executable wrappers, the module will not force early Dispatch.
 
  59    To do this, it uses a handy hack by Thomas Heller for easy access
 
  60    to constants.
 
  61  
 
  62  Typical usage will be::
 
  63  
 
  64    import wmi
 
  65  
 
  66    vodev1 = wmi.WMI ("vodev1")
 
  67    for disk in vodev1.Win32_LogicalDisk ():
 
  68      if disk.DriveType == 3:
 
  69        space = 100 * long (disk.FreeSpace) / long (disk.Size)
 
  70        print "%s has %d%% free" % (disk.Name, space)
