Class VCalendarParser

source code

object --+

        VCalendarParser

Base parser class for iCalendar files. This uses the State pattern (in its Python incarnation, replacing the class of an object at runtime) in order to parse different objects in the VCALENDAR. Other states are

  • VTodoParser: parses VTODO objects.
