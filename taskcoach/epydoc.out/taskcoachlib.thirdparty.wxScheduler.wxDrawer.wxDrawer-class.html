ScheduleSize(schedule, workingHours, firstDay, dayCount)
Static Method

source code 

This convenience static method computes the position and size size of the schedule in the direction that represent time, according to a set of working hours. The workingHours parameter is a list of 2-tuples of wx.DateTime objects defining intervals which are indeed worked. startPeriod and endPeriod delimit the period.
